Materials

  • In order to assign a different material on the face of a mesh from the other faces, go to Edit Mode, select the face, and assign the material.

Use cases

Make a glass material

In order to make a glass material like a window, it looks we can do by setting these things.

  1. Enable alpha-based transparency
    • Go to Material > Settings > Blend Mode
    • Select something like Alpha Blend.
  2. Use Transparent BSDF Node. This is to show objects inside a glass material in Object Mode.
    • Open Shading workspace
    • In Shader Editor
      • Add Transparent BSDF node
      • Add Mix Shader node
      • Connect Shader of Mix Shader to the Surface of Material Output
      • Connect two Shader of Mix Shader to the BSDF of Transparent BSDF and Principled BSDF
  3. Set glass like properties. This is to change a material like a glass and also show images outside of the glass.
    • Go to Material > Surface
    • Set Transmission to 1 or close to it.
    • Set Roughness as you like

See following materials for more details

  • Blender Knowledgebase: Basic Alpha Transparency
  • StackExchange: Glass: Increase "Transmission Roughness" as the object behind goes far
  • Blender 3.0 Manual: Principled BSDF

Troubleshootings

How to find missing texture files and update them?

Go to File > External Data > Find missing files and select a correct directory for texture files.

This solution is from this answer in Stack Exchange.

How to remove images completely from a blender file?

Even if you delete a material using an image file, it can still be selected on a Mateiral. In order to delete it completely, Go to Outliner Editor > Orphan Data Display Mode. Then delete images you want, or click Purge to delete every data, including images.

This solution is from this answer in Stack Exchange, and there is an image in the answer.

3D animation

  1. In order to add an object into your Timeline Editor, choose an object, right click, and click to _Insert Keyframe and select one of them like Location and Rotation.
  2. Set a parent into an object
    1. Click a child object and then click a parent object
    2. Select Object > Parent > Object Now you can see the child object under the parent object in the Outliner editor. You can see the details of parent objects in the manual.
  3. To animate objects based on an arbitary axis instead of the world origin, the origin of an object can be updated.
    1. Select an object you wanna change the axis
    2. Go to Edit Mode
    3. Select Mesh > Snap > Cursor to Selected
    4. Go to Object mode
    5. Select Object > Set Origin > Origin to 3D Cursor Now the object will move, rotate, or scale based on the 3D cursor you set. In order to move the 3D cursor back, Object > Snap > Cursor to World Origin. This is from the answer of StackExchange.

Action

Action is a way to reuse a specific animation of one object. There are two important editors to

  • Dope Sheet > Action Editor mode: To edit an action itself. It's better to start an action from 1st frame
  • Nonlinear Animation: To use multiple actions, animate a scene

To understand an action, see next informative video for how to use Action.

There are a few important things to use Action

  • To change the speed of an action, go to Strip > Action Clip and set the "Playback Scale" value
  • To repeat an action multiple times, go to Strip > Action Clip and set the "Repeat" value

Examples

Door

Modeling

There are a few things to consider before modeling a door

  1. The aspect ratio: According to this article, most of doors have either of following aspect ratios:
    • Two squares: 1:2
    • Golden proportion: 1:1.61
  2. Swing direction: A door is designed differently if it's for a right hand or left hand as well as it is opened to an inward or outward direction. See this article for each case.

Handle

I checked this video to see how to make a model

  1. If you can't choose a Single Vert mesh, go to Edit > Preferences > Add-ons and enable "Add Mesh: Extra Objects" at first
  2. Design a handle using vertexes, so the shape will not be of cylinders.
    1. Select Add > Mesh > Single Vert > Add Single Vert and add the mesh
    2. Extrude the mesh to change the shape of a handle you wanna design
      • Note that it has to be Vertex select mode, not Edge select nor Face select mode to see extrude vertexes
  3. From vertexes, convert them into the set of cylinders
    1. Change the mode to Object mode and select Object > Convert > Curve
    2. Go to Properties Editor > Object Data Properties > Geometry > Bevel and
      1. Increase the Depth
      2. Increase the Resolution
  4. Edit the handle as a mesh and complete modeling
    1. Select the curve and then Object > Convert > Mesh

Note to import 3D models

  1. Make sure that 3D models can be imported into Blender. Some models are exported from other modeling tools and cannot be imported into Blender.
  2. Make sure that the textures are imported correctly. Many of 3D models use textures and they are distributed in a different file sometimes. We need to download them and map textures with those files correctly on Blender.
